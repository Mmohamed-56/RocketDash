## How to play!

- Go to the code section and look at the top right, where it says code; press the drop down and download as a zip.
- Once downloaded extract the folder
- Make your way to the *BUILD* folder and run the **ROCKETDASH.exe** file.
- You may trip your windows firewall, you can just press **More Info** and **Run Anyways**.
- Once you run it you can enjoy it!
